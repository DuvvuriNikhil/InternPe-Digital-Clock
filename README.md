# Digital Clock

A simple Digital Clock application developed using Python and Tkinter. The application displays the current system time in real-time and updates every second through a graphical user interface.

## Features

* Real-time clock display
* Automatic time updates every second
* User-friendly graphical interface
* Lightweight and responsive design

## Technologies Used

* Python
* Tkinter

## How to Run

1. Install Python.
2. Run the following command:

```bash
python digital_clock.py
```

## Learning Outcomes

* GUI development using Tkinter
* Event-driven programming
* Working with time and scheduling functions
